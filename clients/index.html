VRChat Linux Setup Guide

This guide helps fix common Linux / Proton issues with VRChat and walks you through setting up MelonLoader and BepInEx correctly.

ProtonTricks Setup (Fix Native Issues)

Install Steam & ProtonTricks

Arch Linux

sudo pacman -S steam protontricks

Ubuntu / Mint

sudo apt update
sudo apt install steam protontricks

Verify the X: Drive Exists

  1. Open ProtonTricks
  2. Select VRChat
  3. Click OK until you see Choose a wineprefix
  4. Select Select the default wineprefix
  5. Choose Run winecfg
  6. Open the Drives tab
  7. Check if the X: drive exists
    • If missing:
      • Click Add
      • Select X:
      • Apply and close
This fixes the missing drive issue that can cause VRChat to immediately close.

MelonLoader Setup

Download & Install

  • Download MelonLoader: Melonloader.rar
  • Extract the .rar file directly into your VRChat directory

Steam Launch Options (Native Linux)

  1. Open Steam
  2. Right-click VRChat
  3. Select Properties
  4. In Launch Options, paste:
WINEDLLOVERRIDES="version=n,b" %command%

BepInEx Setup

Download & Install

  • Download BepInEx: BepInEx.rar
  • Extract the .rar file directly into your VRChat directory

Steam Launch Options (Native Linux)

  1. Open Steam
  2. Right-click VRChat
  3. Select Properties
  4. In Launch Options, paste:
WINEDLLOVERRIDES="winhttp=n,b" %command%

Notes

  • Make sure VRChat is running under Proton Hotfix
  • If something breaks, re-check the X: drive in winecfg
  • Join the discord https://discord.gg/stellar for updates
