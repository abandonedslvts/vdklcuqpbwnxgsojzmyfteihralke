VRChat Linux Setup Guide

Fix common Linux / Proton issues with VRChat and correctly set up MelonLoader and BepInEx.

ProtonTricks Setup (Flatpak Method)

Install Flatpak (Arch)

sudo pacman -S flatpak

Install Flatpak (Ubuntu / Mint)

sudo apt install flatpak

Install Protontricks (Flatpak)

flatpak install flathub com.github.Matoking.protontricks
Flatpak Protontricks is recommended and avoids system Wine conflicts.

Verify the X: Drive Exists

  1. Run Protontricks: 
    flatpak run com.github.Matoking.protontricks
  2. Select VRChat
  3. Choose Select the default wineprefix
  4. Select Run winecfg
  5. Open the Drives tab
  6. Ensure X: exists
    • If missing → Add → select X:
Missing X: drive will cause VRChat to instantly close.
Warning: We recommend using BepInEx instead of MelonLoader. MelonLoader may not run correctly under Linux / Proton and can cause launch issues or crashes.

MelonLoader Setup

  • Download: Melonloader.rar
  • Extract into your VRChat directory

Steam Launch Options

WINEDLLOVERRIDES="version=n,b" %command%

BepInEx Setup

  • Download: BepInEx.rar
  • Extract into your VRChat directory

Steam Launch Options

WINEDLLOVERRIDES="winhttp=n,b" %command%

Notes

  • Try switching Proton versions if issues persist
  • Re-check the X: drive if VRChat crashes
